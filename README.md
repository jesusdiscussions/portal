# portal

Community comment emailer (Blog Platform Engine).

## GitHub Pages files

Deploy these to your Pages site root (same folder as `posts/`):

- `index.html` — confirm / unsubscribe
- `CommentsPage.html` — comments portal
- `go.html` — optional manual mail handoff helper (not required for normal email links).

**Email links (confirm / unsubscribe / join comments)** go directly to GitHub Pages (`index.html` / `CommentsPage.html`) so GitHub Pages remains the sole public frontend.
